# Main Navigation Test Case

## Title
Verify that the main page displays navigation buttons: Docs, API, Community.

## Preconditions
- User has access to the Playwright website (https://playwright.dev)
- Browser is open and functional
- No prior navigation or actions on the site

## Test Steps
1. Open the browser and navigate to https://playwright.dev
2. Observe the main page content
3. Look for the navigation section or header
4. Check for the presence of "Docs" button/link
5. Check for the presence of "API" button/link
6. Check for the presence of "Community" button/link

## Expected Results
- The main page loads successfully
- The navigation buttons "Docs", "API", and "Community" are visible in the header or navigation area
- Each button is clickable and properly styled
- No errors or broken elements are present
